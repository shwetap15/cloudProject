COURSE PROJECT REPORT

 

 

  • Goal:

 

We will demonstrate distributed Machine Learning using Apache PySpark. We aim at building our own classification algorithms, measuring their performance and comparing it with those provided in PySpark’s MLlib.

 

To achieve this goal, we will analyze the Airline dataset described below and create classification models that will predict whether a given flight would be delayed or not. We will implement our project using the following steps:

 

  1. Data Preprocessing

 

  1. Exploratory data analysis

 

  1. Feature engineering

 

  1. Implementing Machine Learning algorithms for classification by creating ML pipelines:

 

  1. Logistic Regression

 

  1. Stochastic Gradient Descent

 

  1. Naive Bayes

 

  1. Evaluation of the classification models.

 

  1. Comparison of classification models with existing algorithms in PySpark’s MLlib.

 

  1. Displaying results on a webpage.

 

  • Dataset:

 

The dataset, ‘2015 Flight Delays and Cancellations’, has been taken from Kaggle [1]. This is a significantly large dataset with more than a million rows and 31 feature columns which would be a good means to demonstrate PySpark’s abilities to tame Big Data and create machine learning pipelines.

This dataset is collected from the U.S. Department of Transportation’s (DOT) Bureau of Transportation Statistics which contains information of flights in the year 2015 with respect to airline carrier, original airport, destination airport, distance traveled by the flight, time spent, departure delay time, arrival delay time, etc.

 

  • Context:

 

With the booming airline business, statistics of flight delays have become an important factor in understanding the airline’s performance. Delay in flights causes a loss to both the airline and the passengers in different ways. The airline’s reputation is at stake and the passengers face loss in terms of time and money. There are different factors for flight delays that are available in this dataset and they can be analyzed to find a pattern and predict if a certain flight will be delayed. We feel that creating predictive models for this would help as a starting point for the airline take measures in understanding flight delays.

 

Analyzing such a huge dataset by leveraging machine learning models on PySpark will help us understand handling large scale data and extract patterns for data

 

analytics. PySpark is also efficient in processing data in real-time which is an important industry skill to know.
